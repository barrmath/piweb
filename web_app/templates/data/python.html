Introduction

Python est un langage de programmation interprété. Ainsi, il est indépendant de la machine tant que vous pouvez installer python sur celle-ci. Par contre, vous devez installer python. En tant que langage interprété, (et non compilé), il a des performances en retrait par rapport a du C par exemple.
Il a comme avantage d'être simple, et relativement intuitif.
