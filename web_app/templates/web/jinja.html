Jinja2

Flask est fourni par défaut avec Jinja2 comme moteur de template.
Vous pouvez utiliser un autre moteur si vous voulez : par exemple angular.

Le but de Flask et de jinja est de permettre de faciliter le travail du développeurs. Jinja2 vous permets de générer du code html à partir de certains objet.
exemple : faire une page de base qui se repete dasn tout le site (les menus, la disposition de la page ...)

Réutiliser du code html

Alors Jinja2 posséde des balises un peu comme html :
  • les balises {{ truc }} : permettent d'accéder à un objet
  • les balises {% machin %} : permettent de lancer des fonctions
