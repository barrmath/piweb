Git

Qu'est-ce que Git

Git est un logiciel de gestion de versionning. C'est à dire qu'il permet de revenir en arriére en cas de soucis. Bref une sorte de super sauvegarde.

Git est souvent associeé à GitHub ou Gitlab, qui permette de sauvegarder des projets sur le cloud.

C'est un outil qui est transveral au niveau du projet.

  • Les développeurs vont utiliser git pour sauvegarder leurs travails et faire plusieurs version
  • Les administrateurs vont utiliser git pour pouvoir déployer les projets
  • Les chefs de projets peuvent surveiller l'avancement des projets

Pour faire simple Git est un indispensable dans le monde du développement actuel.

Au niveau de l'installation de git, sous linux utilisez votre gestionnaire de paquets (souvent Git est déjà installé). Pour les windosiens : https://git-scm.com/downloads/win

Vous avez en général 2 repos (ou repository ou en français dossier) le repos local ou vous travailler et le repos remote ou cloud qui est distant.
