HTML

Pourquoi apprendre le HTML (et le CSS) en 2024? Wordpress, Ionos font des sites internets en no code.
Je vais donner plusieurs raisons:
  • Si voulez un site leger pour rappel une page web fait la taille de doom.
  • Vous aimer appliquer le KISS (Keep It Simple and Stupid)
  • Vous trouvez pas votre bonheur dans les wordpress et autres
  • Vous avez besoin de récupérer des données sur des sites web (webscrapping)
Commençons par le commencement le html puis ensuite le CSS.

HTML

Premierement pour apprendre à faire des pages internets en pure HTML CSS vous avez besoin d'un navigateur internet et d'un editeur de texte.
N'ouvrez pas word ou autre suite bureautique, il nous faut du texte sans fioriture.
Pour les navigateurs internets, je recommande firefox et le firefox pour developpeur pour aller plus loin
Pour les éditeurs de texte :
  • Éditeur simple : notepad++
  • Éditeur pour developpeur : vscodium

Ouvrez un éditeur de code, copier/coller le code en dessous et sauvegarder avec une extension .html.
<!DOCTYPE html> <html lang=""> <head> <meta charset="utf-8"> <title></title> </head> <body> <header></header> <main>Hello World</main> <footer></footer> </body> </html> Ensuite ouvrez le fichier avec votre navigateur.

Félicitation vous voilà développeur.
Analysons un peu le code. On vois des balises ouvrantes < mot > et fermante < /mot >.
La premiere balise <!DOCTYPE html> indique simplement qu'il s'agit d une page HTML.
La deuxieme balise ouvre le code et vous pouvez indiquer une langue.
Ensuite la balise head permet d'afficher des choses en dehors la page comme le titre en haut. et vous pouvez mettre des metadonnée grâce à la balise meta
charset="utf-8" permets de specifier le jeu de caractere utiliser. (a ne pas confondre avec la police)

On arrive sur la balise body qui indique que vous travailler sur le corps de la page html.
  • head pour le haut de la page
  • main pour le principal article
  • footer pour le bas de page
En fonction de ce que vous voulez faire, il y a plein de balise en html. Je vous laisse les découvrir ici Malgrès les balises HTML, vous trouver que votre page manque de couleur, vous voudrez avec plus de choix en placement. Le CSS est fait pour ça.

Le CSS

Il faut lier le fichier CSS dans la page internet. <link href="style_base.css" rel="stylesheet"> Maintenant regardons ce que fait ce fichier,
Prenons par exemple la partie footer du css de ce site (qui auras peut être changer entre 2): footer{ background-color: #333; padding-bottom: 100%; border-top:solid #8fd9fc; } Vous avez donc le nom de la balise à modifier des accolades puis des attributs (oubliez pas le ";" à la fin).
Ici le fond est noir, on prends toute la place en bas (bottom) et on crée une ligne "solid" avec son code couleur.
Si vous avez plusieurs balises amis que vous en voulez pas avoir le meme comportement pas de soucis. Ajouter class="nom" (si vous avez plusieurs balises à modifier) ou id="nom"(pour une seule balise)
Par exemple <pre class="Code">blablabla</pre> La partie CSS suivante :
.code{ background-color: #000; color: rgb(75, 255, 120); padding-left: 10px; padding-right: 10px; } La différence : le point . avant le nom indique une classe ou une id.
Vous voilà dévelepoppeur front end. (enfin presque).
